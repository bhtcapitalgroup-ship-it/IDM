# Agentic Company Builder

## Mission
Build a controlled agent orchestration platform that can create, manage, and coordinate specialist agents to help build and operate a trader evaluation business.

## Core Principles
- Modularity
- Auditability
- Security
- Clear boundaries
- Explicit workflows
- Human approval for sensitive actions

## Non-Negotiables
- No unrestricted production access for agents
- No autonomous destructive actions
- No autonomous production deployment
- Human approval required for billing, payout, legal, and permission escalation changes
- All critical actions must be logged

## Initial Agent Roles
- Executive Orchestrator
- Product Architect Agent
- Frontend Builder Agent
- Backend Builder Agent
- Database Builder Agent
- QA Inspector Agent
- DevOps Operator Agent
- Compliance Reviewer Agent

## Initial Core Systems
- Agent Registry
- Task Registry
- Workflow Engine
- Prompt Registry
- Tool Registry
- Memory / Context Layer
- Approval Layer
- Audit Logs
- Role / Permission System

## Preferred Stack
- Next.js
- React
- Tailwind
- shadcn/ui
- FastAPI
- PostgreSQL
- Redis

## Engineering Style
- Small composable modules
- Clear naming
- Strong boundaries
- Explicit schemas
- Validation at system boundaries
- Basic error handling from the start
- Production-minded but minimal v1
